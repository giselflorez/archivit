## TRANSCRIPTION SETUP

**Tools Required:**
```bash
# Option 1: OpenAI Whisper (recommended)
pip install openai-whisper

# Option 2: Whisper.cpp (faster, local)
brew install whisper-cpp

# Option 3: FFmpeg for audio extraction
brew install ffmpeg
```

**Transcription Command (when tools installed):**
```bash
# Using Whisper
whisper /path/to/video.mp4 --model base --output_format txt

# Batch transcription
for f in /Users/onthego/ARCHIVIT_01/dataland_proposal/media/*.mp4; do
  whisper "$f" --model base --output_dir /Users/onthego/ARCHIVIT_01/dataland_proposal/transcripts/
done
```
