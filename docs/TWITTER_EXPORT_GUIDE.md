# How to Export Your Twitter/X Archive

## Download Your Archive

1. **Go to Twitter/X Settings:**
   - Visit: https://twitter.com/settings/download_your_data
   - Or: Settings → Your Account → Download an archive of your data

2. **Request Your Archive:**
   - Click "Request archive"
   - Twitter will prepare your data (takes 24-48 hours)
   - You'll get an email when it's ready

3. **Download the ZIP File:**
   - Click the link in the email
   - Or go back to: https://twitter.com/settings/download_your_data
   - Click "Download archive"
   - Save the ZIP file

4. **Extract the Archive:**
   - Unzip the file
   - Look for: `data/tweets.js` or `data/tweet.js`

## Quick Alternative: Recent Tweets Only

If you want to start immediately without waiting for the full archive:

- Use Twitter's API (requires developer account)
- Or manually copy/paste recent tweets into a text file
- Save as: `~/Desktop/WEB3AUTOMATE_UPLOAD/recent_tweets.txt`

## What's Included

Your archive contains:
- All your tweets
- Retweets
- Replies
- Likes
- Direct messages
- Profile information
- Account history

## Next Steps

Once you have the archive:
1. Extract the ZIP file
2. Run the Twitter processor script
3. Your tweets will be added to your knowledge base
