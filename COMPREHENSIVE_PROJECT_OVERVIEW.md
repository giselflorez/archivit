## Agent Analysis Pipeline (8 Stages)

1. **INGEST** - Accept file/URL, validate, create source record
2. **TRANSCRIBE** - Whisper ASR, detect language, confidence scores
3. **DIARIZE** - Speaker identification, match against Masters bank
4. **SEGMENT** - Topic boundaries, key passages, quotable segments
5. **EXTRACT** - NER (people, places, dates), claim detection, sentiment
6. **CROSS-REFERENCE** - Match to knowledge bank, find supporting/contradicting
7. **INDEX** - Embeddings, full-text search, topic clusters, knowledge graph
8. **PRESENT** - Display with confidence indicators, enable verification
